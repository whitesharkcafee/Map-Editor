# Map Editor
(insert logo here)

Map Editor is a mod for Haze Games' game "Fractal Space" that adds a long-awaited "Level Editor" functionality to the base game. Inspired by Jav's Level Editor and BEEMod, Map Editor aims to build a modular system that allows players to create not only their own maps, but also items for others to play with, making it easy to keep the mod alive without even touching the code.

Main features (will grow in future):

 1. Modular system for objects - let players add their own objects with a simple asset bundle and a json file.
 2. Real-Time Playtesting - system first seen in [official Level Editor footage](https://www.youtube.com/watch?v=ASMXYud0uS8)
 3. No more garbage from Chapter 4 - Map Editor uses Modding Scene, completely empty, no more slow loading and horrible performance.
 4. No more JSON - Map Editor uses binary serialization/deserialization for saving the levels (for now it's JSON, in future, it will be binary), allowing for even better performance.
